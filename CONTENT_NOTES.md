Design takeaway:

- Keep the site more premium than a plain student portfolio.
- Avoid overly dense layouts; the profile should breathe.
- Prioritize mobile performance and responsive behavior over heavy visual effects.
- Use subtle motion such as logo/certificate marquee, floating proof cards, hover states, and clean section transitions.
- Preserve Astro-style static performance with minimal runtime JavaScript.
- Use Thai/English content architecture clearly, with hreflang/canonical intact.
